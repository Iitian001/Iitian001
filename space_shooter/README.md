## 🎮 Play the Game

No installation required! Just download the repository and open `index.html` in any modern web browser to start playing.

## ✨ Features

- **Endless Action:** Survive wave after wave of increasingly difficult enemy machines.
- **Epic Boss Fights:** Test your skills against massive, heavily armed boss ships.
- **Neon Aesthetic:** Gorgeous retro-wave visual style with glowing particles, dynamic shadows, and a parallax starfield.
- **Fluid Controls:** 360-degree aiming, responsive movement, and a life-saving dash mechanic.
- **Dynamic Audio:** Custom sound effects synthesized entirely from scratch using the Web Audio API—no external audio files needed!
- **Power-Ups:** Collect upgrades to enhance your arsenal and restore health.

## 🕹️ Controls

| Action | Input |
| :--- | :--- |
| **Move** | `W` `A` `S` `D` or `Arrow Keys` |
| **Aim** | `Mouse Cursor` |
| **Shoot** | `Left Mouse Button` (Click or Hold) |
| **Dash** | `Spacebar` |

## 🛠️ Tech Stack

NEON VOID is a testament to what can be achieved without heavy engines or frameworks. 
- **HTML5 Canvas:** For high-performance 2D rendering.
- **Vanilla JavaScript (ES6+):** Custom game loop, entity management, and collision detection.
- **Web Audio API:** Procedural generation of synth sound effects (Square, Sine, and Sawtooth waves).
- **CSS3:** For UI overlays and glowing HUD elements.

## 🚀 How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/neon-void.git
   ```
2. Navigate to the folder:
   ```bash
   cd neon-void
   ```
3. Open `index.html` in your favorite browser. (Alternatively, use a local server like VS Code's Live Server for the best experience).
